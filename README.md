基于neovis.js的一个neo4j以太坊网络数据展示用前端，使用传统方式开发

集成网络拓扑图显示，最高度节点查询，节点信息查找

以及通过 Neo4j Graph Data Science (GDS) 库实现计算节点拓扑图的四种中心性指标：（PageRank、度中心性、亲密中心性、介数中心性）

集成seed-emulator-ETH搭载以太坊仿真器的相关页面
